To Run the Client - Server Communication
The program is set to run on localhost port 9001

1) Change the "hostname" in REMS_Testmain.java to "localhost"  if it's not already localhost
2) Change the file path in REMSClient.java  to your local workspace
1) Run REMS_Testmain.java From src > REMS > REMS_testFramework > REMS_Testmain.java


While importing the project from github if you encounter exclamation mark then 
1) right click on your project > properties > build path > add Library > jre system Library


NOTE: The project uses jre_1.8.
